# Lottery Probability Calculator

A Python project to calculate win probabilities for lottery tickets based on historical data.

## Features
- Supports 4-digit, 5-digit, 6-digit, and 8-digit lottery combinations
- Analyzes historical data (last 1 year, daily draws)
- Calculates number repeat frequencies and time intervals
- Generates empirical probabilities for each number
- User can input ticket number to get win probability

## Installation
1. Clone or download the project.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python lottery_calculator.py`

## Usage
- Generate sample data: `python generate_data.py`
- Run calculator: `python lottery_calculator.py`
- Input your ticket number and select digit length to see probability.
